## 一键安装
```
curl -L https://raw.githubusercontent.com/honjow/GPD-WinControl/main/install.sh | sh
```

## 功能
- 键鼠模式的按键映射配置
- 摇杆死区和边界调整
- 震动强弱调整
- RGB灯光控制
